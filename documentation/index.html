Chassis Web Components

  1. chassis-cycle: For wizards, tab navigation, scrollers, or anything else that needs to display one thing at a time.
  2. chassis-layout: A quick way to create a flexbox-based page layout. Great for apps.
  3. chassis-list: A powerful and simple list management component.
  4. chassis-overlay: Creates an overlay, like the mask behind a modal window or a loading window screen.
  5. chassis-dialog: A popup pane.
